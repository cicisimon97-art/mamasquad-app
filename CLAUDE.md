# MamaSquads App

## What This Is
MamaSquads is a verified, moms-only playdate community app. Tagline: "Less Chaos. More Playdates."
It helps mothers find and organize playdates with verified moms in their area.

**Live site:** https://mamasquads.com

## Tech Stack
- **Frontend:** React 18 + Vite (single large `src/main.jsx` ~6600 lines)
- **Backend:** Supabase (auth, database, RLS)
- **Hosting:** Vercel
- **Native wrapper:** Capacitor (for iOS App Store)
- **Styling:** Inline styles (no CSS framework)
- **Theme:** Primary `#6B2C3B` (burgundy), Background `#FFFBFC`

## Key Files
- `src/main.jsx` — Main app component (all screens, state, logic)
- `src/app.jsx` — Additional app utilities
- `src/supabaseClient.js` — Supabase initialization
- `supabase-schema.sql` — Full database schema
- `capacitor.config.json` — Capacitor/iOS config
- `ios/` — Xcode project (generated by Capacitor)
- `public/manifest.json` — PWA manifest
- `public/sw.js` — Service worker

## Commands
- `npm run dev` — Start dev server
- `npm run build` — Production build
- `npm run build:ios` — Build + sync to iOS
- `npx cap sync ios` — Sync web assets to iOS project
- `npx cap open ios` — Open Xcode project (requires Xcode)

## App Store Deployment (IN PROGRESS)
The app is being wrapped with Capacitor for iOS App Store submission.

### What's Done (on the dev laptop without Xcode):
- Capacitor installed and configured
- iOS platform added (`ios/` directory)
- App ID: `com.mamasquads.app`
- Build scripts configured

### What Needs to Be Done (on a computer WITH Xcode):
1. `git pull` to get latest changes
2. `npm install` to install dependencies
3. `npm run build:ios` to build and sync
4. `npx cap open ios` to open in Xcode
5. In Xcode:
   - Set your Apple Developer Team under Signing & Capabilities
   - Set the deployment target (iOS 16+ recommended)
   - Configure app icons (use the logo.png in `public/`)
   - Test on a simulator or device
6. Archive and upload to App Store Connect
7. Fill out App Store listing (screenshots, description, etc.)

### Apple Developer Account Required
You need an Apple Developer account ($99/year) at https://developer.apple.com/programs/
